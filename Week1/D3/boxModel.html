Box model

El box model en CSS describe cómo se visualiza y se comporta un elemento en la página web.

El contenido tiene el fondo azul.

El relleno del padding está en verde

El borde está en rojo

El margin es la distancia entre el borde de la caja y el resto de elementos de la web.

Acuérdate de usar el inspector (F12) para revisar las distancias.

Posicionamiento

El posicionamiento en CSS determina cómo se ubican los elementos en relación con su entorno.

Este es un párrafo normal hecho en span. Elemento posicionado de forma relativa, usando span. Mira cómo afecta el posicionamiento relative a otros elementos. Aunque el texto relative (en verde) se haya desplazado, la web reserva su espacio original.





Este es un párrafo normal hecho en span. Elemento posicionado de forma absolute, con span. Fíjate como, pese a que el span anterior sea absolute, aparece muy lejos de su posición original. Absolute se mide desde el punto 0,0 de la web (arriba a la izquierda). La web no reserva el espacio de un bloque absolute.
