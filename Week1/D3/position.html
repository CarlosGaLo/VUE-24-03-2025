Este es un párrafo normal hecho en span. Elemento posicionado de forma relativa, también es un span Mira cómo afecta el posicionamiento relative a otros elementos. Aunque el texto relative (en azul clarito) se haya desplazado, la web reserva su espacio original.
Este es un párrafo normal hecho en span. Elemento posicionado de forma absoluta, es un span. Fíjate como, pese a que el span anterior sea absolute, aparece muy lejos de suposición original. Absolute se mide desde el punto 0,0 de la web (arriba a la izquierda). La web no reserva el espacio de un bloque absolute.
