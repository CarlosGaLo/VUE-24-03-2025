Ejemplos de Transformaciones y Efectos en CSS

Rotate

transform: rotate(30deg)

Al pasar el cursor por encima de este recuadro, verás un giro de 30 grados.

Translate

transform: translate(30px, 30px)

Este recuadro se desplazará 30px en ambos ejes X e Y cuando pases el cursor sobre él.

Scale

transform: scale(2)

Al situar el cursor, el recuadro se duplicará en tamaño.

SkewX

transform: skewX(50deg)

Este recuadro se inclinará (sesgará) 50 grados en el eje X al pasar el cursor.

Sombra

box-shadow

Este recuadro aplica una sombra más intensa. Puedes modificar la propiedad box-shadow para variar la posición, el difuminado y la intensidad de la sombra.

Opacidad

opacity

Al pasar el cursor, la opacidad de este recuadro desciende al 50%.

Nota: Recuerda que rotate, translate, scale y skewX se aplican a través de la propiedad transform. Por otro lado, box-shadow y opacity se establecen de forma independiente.
