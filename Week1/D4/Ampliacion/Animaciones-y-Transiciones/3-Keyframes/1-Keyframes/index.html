@Keyframes en CSS

La regla @keyframes en CSS se utiliza para definir los cambios de estilo que tendrá un elemento en diferentes etapas de una animación. Se combinan con la propiedad animation (y sus propiedades asociadas, como animation-duration, animation-timing-function, animation-delay, etc.) para dar vida a transiciones dinámicas en los elementos de una página web.

La sintaxis básica de @keyframes es: 


@keyframes nombreAnimacion {
  0%   { /* Estilos al inicio de la animación */ }
  50%  { /* Estilos a la mitad de la animación */ }
  100% { /* Estilos al final de la animación */ }
}

Una vez definido el nombre de la animación con @keyframes, se asocia a un elemento con: 


.elemento {
  animation-name: nombreAnimacion;
  animation-duration: 2s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  /* ... y otras propiedades de la animación ... */
}

De forma abreviada, se puede utilizar la propiedad animation: 


.elemento {
  animation: nombreAnimacion 2s infinite alternate;
}
